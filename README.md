# [《鹦鹉学会了一句话》](https://jinrihlbl.github.io/)

林婉的鹦鹉只会说一句话：“等等，就快好了。”

那是丈夫李国栋的口头禅。结婚三十年，这句话成了他们生活的背景音。煎鱼时油锅在响——“等等，就快好了。”她催他修漏水的水龙头——“等等，就快好了。”甚至在她母亲去世那晚，她蜷缩在沙发哭泣，他搓着手站在门口：“等等，就快好了。”

她始终没明白，悲伤这种事，怎么会“快好了”。

鹦鹉是儿子送的。去年儿子搬去另一个城市，怕她寂寞，买了这只绿色的虎皮鹦鹉。“妈，它会学说话，你教它。”儿子说这话时，行李箱就立在门边。

林婉给鹦鹉取名“等等”。很贴切。

她试着教它说“你好”“吃饭了”，甚至哼唱年轻时常唱的歌。但鹦鹉沉默如石，只是歪头看她，黑眼睛像两粒深井。直到三个月前的雨夜。

那晚风湿痛发作，她靠在旧藤椅上敷热毛巾。电视机开着，新闻里说今年梅雨季特别长。她忽然对着空气喃喃：“国栋，帮我倒杯水好吗？”

沉默。

然后，从阳台角落传来清晰的声音：“等等，就快好了。”

她差点打翻水杯。鹦鹉站在栖木上，字正腔圆，连丈夫那种特有的、尾音微微上扬的腔调都一模一样。仿佛他从未离开——去年秋天，他在睡梦中安静地走了，像一片叶子落进水里。

从此鹦鹉只重复这一句。清晨她撒鸟食时，它说：“等等，就快好了。”午后她打盹时，它说：“等等，就快好了。”深夜她望着丈夫遗像发呆时，它还是那句：“等等，就快好了。”

邻居王阿姨来做客，听了直皱眉：“这话听着怪难受的。你教它说点别的吧？”

林婉摇头。她开始明白，鹦鹉不是在学舌，是在替某种未完成的东西延续生命。

昨天收拾衣柜，她翻出一件丈夫的旧毛衣。肘部磨破了，她曾多次说要补，他总是：“等等，就快好了。”现在毛衣还在，破洞还在，说这话的人却永远不会来补了。

她把毛衣摊在膝上，忽然想起婚礼那天的情景。敬酒时他西装口袋里藏了手帕，偷偷帮她擦汗，伴郎起哄，他红着脸说：“等等，就快好了——我是说婚礼流程快好了。”宾客大笑。

原来这句话贯穿了他们的一生。年轻时是甜蜜的搪塞，中年时是疲惫的托词，晚年成了真实的写照——死亡确实“快好了”，对所有人都是。

今天早晨，她决定教鹦鹉说新的话。

“早安。”她对它说。

鹦鹉梳理羽毛。

“今天天气不错。”

鹦鹉喝水。

她坚持不懈。做午饭时对着它念叨菜谱，洗衣服时哼歌，甚至翻出相册一张张讲解：“这是结婚照”“这是儿子满月”……

鹦鹉始终沉默。

黄昏时她累了，坐在阳台摇椅上。夕阳把防盗窗的影子拉长，一格一格印在她身上，像被困在某个巨大的表格里。她轻轻说：“国栋，我有点累了。”

然后，她听见了。

不是鹦鹉的声音。是记忆深处传来的，三十年前那个春日午后，他在装修婚房，满身石灰粉，转身对她笑：

“婉婉，再等等，我们的好日子就快好了。”

眼泪毫无预兆地涌出。原来鹦鹉复述的从来不是语言本身，而是语言背后的时间。那些被“等等”拖延的瞬间，那些在“就快好了”中流逝的岁月，统统凝结成这一句话，像琥珀困住昆虫。

她走到鸟笼前，打开小门。

鹦鹉歪头看她，没飞出来。

“你自由了。”她说。

鹦鹉扑棱翅膀，落到她肩上，用喙轻轻碰她耳垂，然后说：“等等，就快好了。”

这次她听懂了。不是敷衍，是承诺——丈夫在生命尽头未能说出口的话：离别不会太久，我们“就快”重逢。

她笑了，把鹦鹉放回笼子，添了清水和谷粒。

窗外，华灯初上。她知道明天、后天、大后天，鹦鹉依然会说那句话。而她终于学会聆听这句话里的全部含义：它既是拖延，也是期盼；既是遗憾，也是希望；既是未完成，也是永恒。

夜风拂过，鸟笼轻轻转动。她对着空荡的客厅说：

“好，我等你。”
